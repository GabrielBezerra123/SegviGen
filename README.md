## 🛠️ Troubleshooting

If SegviGen does not open, try these steps:

- download the file again

- run the app as administrator
- restart Windows and try again
- make sure your graphics drivers are up to date

If the app opens but the 3D model does not load:

- check the file format
- try a smaller model
- move the model file to a simple folder path, such as `C:\3D\model.obj`
- remove special characters from the file name
- test with another 3D file

If the app feels slow:

- close other apps
- use a smaller model
- check free RAM
- make sure your GPU drivers are current
